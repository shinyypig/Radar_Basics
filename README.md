# 雷达信号处理中的矩阵方法

本书以雷达信号处理为主线，从基础雷达原理出发，逐步引入矩阵、子空间、张量和优化等工具，强调如何用统一的矩阵视角理解信号建模、检测、估计、跟踪与定位问题。内容安排兼顾理论推导与工程直觉，适合作为雷达信号处理入门到进阶的课程讲义或自学材料。

## 本书大纲

全书按照“基础理论 -> 信号模型 -> 检测与估计 -> 跟踪与定位”的思路展开：

1. 先介绍雷达的基本概念、系统组成和工作原理，建立全书背景。
2. 然后补充矩阵分解、张量表示等数学基础，为后续算法做准备。
3. 在此基础上讨论雷达发射与接收信号模型，明确距离、角度、多普勒等参数在数据中的体现形式。
4. 接着进入目标检测与参数估计，介绍脉冲压缩、积累、波束形成、子空间方法等经典技术。
5. 最后讨论目标跟踪与无源定位，展示矩阵方法在更复杂任务中的应用。

## 章节简介

### 绪论

介绍雷达的发展背景、基本概念、系统组成、典型工作频段以及距离、速度、角度等基本探测机理，为全书后续内容建立统一的问题背景。

### 矩阵方法基础

回顾特征分解、奇异值分解、低秩表示、稀疏分解和张量等核心数学工具，说明这些方法为什么适合描述雷达中的结构化数据与算法问题。

### 雷达发射与接收信号模型

从单天线脉冲雷达出发，介绍发射信号、回波信号以及线性调频等典型波形模型，并进一步过渡到阵列、多脉冲等更高维观测形式。

### 雷达信号检测

围绕目标是否存在这一核心问题，介绍脉冲压缩、匹配滤波、相干与非相干积累、检测门限以及恒虚警率检测等基本方法。

### 目标参数估计

讨论如何从接收数据中估计目标的方位、速度等参数，内容涵盖经典波束形成、最小方差方法、子空间方法以及稀疏和张量视角下的估计思路。

### 目标跟踪

介绍目标状态估计与数据关联两大核心问题，讨论如何利用连续时序观测对目标运动状态进行递推更新，并在多目标和杂波环境下维持稳定航迹。

### 无源定位

介绍不依赖主动发射的定位思想，重点讨论测向定位、时差定位和频差定位等方法，并展示如何将定位问题转化为矩阵方程或优化问题求解。

### 附录

补充本书使用的符号约定、矩阵函数、卷积与相关的矩阵表示、向量范数与矩阵范数以及张量乘法性质等基础知识，方便查阅。
